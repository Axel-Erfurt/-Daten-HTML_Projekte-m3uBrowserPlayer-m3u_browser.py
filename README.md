# m3uBrowserPlayer
play m3u TV-Playlist in browser

### Requirements

- python3

### Usage

```
cd path/to/m3uBrowserPlayer
```
```
python3 m3u_browser.py input.m3u
```

or

```
python3 m3u_browser.py input.m3u output.html
```
